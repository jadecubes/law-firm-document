# API Documentation

This document provides an overview of the API documentation available for the Law Firm Admin Provisioning system.

## Admin API

**API Spec**: `/examples/admin-api.yaml` (OpenAPI v3.1.0)
**Version**: 1.3.0
**Title**: Law Firm Admin Provisioning API (Logto-managed RBAC)

The Admin API provides comprehensive endpoints for:
- Law firm management
- User provisioning
- Logto organization management
- Resource access grants
- Support access sessions
- Capabilities and policies

### Documentation Locations

1. **OpenAPI Reference**: [Admin API Documentation](/docs/admin-api/law-firm-admin-provisioning-api-logto-managed-rbac)

2. **BDD Feature Specifications**: [BDD Features](/docs/features/)
   - Detailed behavior specifications for each endpoint
   - Given/When/Then acceptance criteria
   - User stories and scenarios
   - Organized by feature area:
     - Access Grants
     - Capabilities
     - Firms
     - Logto Bridge
     - Support Access
     - Users
